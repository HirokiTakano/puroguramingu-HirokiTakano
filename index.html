大学で学んだ言語

Java

私は大学に入るまでプログラミングを学んだことがなかったため、大学に入ってから授業で扱ったJavaが初めてプログラミングに触れた言語です。

そこから、ずっと大学ではJavaを学んでいたので現時点でJavaを学習し始めてから２年程になりますが、大学から与えられていた課題をこなしていただけだったので、課題に対するコードは作成してきたのですが、自分でJavaを用いて何かを作成するといったことはまだしていません。



独学で学んだ言語


HTML & CSS

独学で学んだ言語としての１つ目はHTML & CSSです。

この言語を学んだ理由としては自分でサイトを運営したり、他者に何かを伝えるにしても自分のコードを公開することに対しても今後絶対に必要になってくるスキルだと思ったからです。

HTML & CSSを学習し始めた時期としては約半年前からです。私はjavascriptやPHPのスキルがまだないため、サイトにHTMLとCSSのみを用いてサイトを作成することしか出来ませんが、この技術のみのサイト作成であれば自分一人で作成することが可能です



Unity C#

自分は高校生の頃から一度は自分でゲームを作成してみたいと思っていたので興味本位のみで動画やサイトを見て、学ぶことよりもゲームを完成させるためにこの言語に触れたので他の言語よりも理解度としては低くなっていますが、自分で作成したものが以下の２つです。

Youtubeリンク➢ break brocks

1つ目の作成物は『break brocks』というゲームです。

このゲームはプレイヤーが矢印キーを用いて下の長方形のブロックを左右に動かすことでボールを反射させて、ブロックを壊すゲームになっています。


Youtubeリンク➢ Roll a Ball

2つ目の作成物は『Roll a Boll』というゲームです。

こちらのゲームも先ほどと同様にプレイヤーは矢印キーを用いることで操作することが出来ます。このゲームではボールを動かすことによって、浮いている物体を取得していくという内容になっています。そして、かなり見づらくはなってしまっていますが左端にカウントが表示されていて自分が何個取得したかが分かるようになっています。

以上が私のプログラミング遍歴になります。
